My Personal Projects

Artists vs Cubeman

Artists vs Cubeman is an Android game currently under development. Its game mode is a mix between action and puzzle, the objective is to paint setted little 5x5 tile images before the time run outs, but using your enemies as paint!

Core Dump Wars

Core Dump Wars is a spaceships game written fully on assembler using emu8086.

Who I am

Who I am is a fully responsive HTML5/CSS3 web application designed to let the people know about me and my projects.

Hexagonal
