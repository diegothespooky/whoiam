Artists vs Cubeman

An independent Android game

Use your enemies as paint!

Artists vs Cubeman is a mix between action an puzzle, the objective is to paint setted little 5x5 tile images before the time run outs, but using your enemies as paint!

During each level the complexity of the images will be increasing and also the speed, so it will add the challenge factor to produce an entertaining-to-addictive game.

Invite a friend or challenge yourself on different game modes, and earn game coins to unlock different characters and stages.

This game is currently under development by Diego López.

Stay tuned for updates and news!

Pixel Art + Android

This project combines two of my passions: Art and Software Development. I started this project with a few ideas in my head and a sketchbook, and now is all integrating on an Android apk.

Android Canvas!

This game is developed fully with the Android SDK, written on Java using the android.graphic.Canvas class to render frame by frame animations of bitmaps designed with the pixel art software Aseprite.
