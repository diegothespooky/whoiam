Core Dump Wars

A spaceships game written in assembler using emu8086.

MOV spaceship,AX

This is an spaceships game written, designed, painted and developed by me on the year of 2009.

The objective of the game is to travel trough the galaxy dodging and destroying asteroids until you reach the enemy spaceship wich you must defeat using your cannon

This application is written fully on x86 assembly language, using various system calls for printing bitmap graphics and playing sounds. The development of this application served to me for research and learning purposes and now I publish it so it may serve others as well for learning and reference

Its fully functional and available at my github page, feel free to clone it, fork it, downloading it or watherver you need and send me any feedback.
