Hexagonal

A set of responsive SVG based Web Components.

SVG Web Components

Hexagonal is a set of fully responsive, HTML5/CSS3 Web Components. All the components are based on SVG graphics to made them light weight and more dynamic since each object or polygon can have its own events.

All the components are easy to include and customize and are written on coffeescript and sass. All the components are beign tested to be compatible with bootstrap and skel.

For the development I am using grunt as the task runner with jshint and jasmine plugins to achieve high quality and bug free code.

This project is still under development, contact me for more information.

You can also take a look at the development progress on the github repository.
