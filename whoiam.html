Who I am

My personal website.

You are living it right now!

"Who I am" is an HTML5/CSS3 fully responsive web application designated to show the people information about me and my projects.

The initial design and scaffolding was created by HTML5 UP using skel and some pretty cool libraries like font awesome and jquery.

Through time I've been adding some other nice jquery libraries like textillate and lettering to add a touch of creativity to the page, and I will add some more soon.

If you have any suggestions to make this website nicer please don't hesitate on contacting me. You can also take a look at the code on github on the link below.
